When you run the analysis, you'll get these visualizations:

1. monte_carlo_trade_shuffle.png
   ├─ Histogram of shuffled trade results
   ├─ Actual performance vs random distribution
   └─ Statistical significance marked

2. monte_carlo_shuffle_cdf.png
   ├─ Cumulative distribution function
   ├─ Probability of achieving each return
   └─ Percentile analysis

3. monte_carlo_bootstrap.png
   ├─ Distribution of resampled equity curves
   ├─ 5th-95th percentile confidence intervals
   └─ Risk/reward assessment

4. monte_carlo_qq_plot.png
   ├─ Q-Q plot vs normal distribution
   ├─ Tests for fat tails
   └─ Tail risk assessment

5. monte_carlo_summary.png
   ├─ All methods in one view
   ├─ Comparison table
   └─ Quick reference dashboard
